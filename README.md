## 1. 프로젝트 개요

Truve는 **XRPL(XRP Ledger) 테스트넷** 위에서 동작하는 기부 플랫폼 프로토타입입니다.  
기부자는 재단을 탐색하고 XRP로 기부하며, 트랜잭션 해시를 통해 기부금 사용처를 온체인으로 직접 추적할 수 있습니다.

### 핵심 가치

| 가치 | 설명 |
|------|------|
| **투명성** | 모든 기부 트랜잭션을 XRPL에 기록, 누구나 조회 가능 |
| **분산 기부** | ETF 뮤음 기능으로 여러 재단에 한 번에 비율 기부 |
| **커뮤니티** | 기부자끼리 토론·랭킹·묶음 추천으로 집단 지성 활용 |
| **거버넌스** | 기부자 투표로 플랫폼 운영 방향 결정 |

## 2. 디자인 시스템

### 색상

| 용도 | 값 |
|------|----|
| 배경 | `#f2f2f2` |
| 카드 / 헤더 | `#ffffff` |
| 주요 텍스트 | `#1a1a1a` |
| 보조 텍스트 | `#777`, `#aaa` |
| 액센트 (활성 탭·버튼) | `#1a1a1a` (검정) |
| 번들 액센트 | `#6366f1` → `#8b5cf6` (인디고·퍼플 그라디언트) |
| 위험/좋아요 | `#e05a00`, `#ff4646` |

### 카테고리 배지 색

| 카테고리 | 배경 | 텍스트 |
|----------|------|--------|
| 환경 | `#dde9ff` | `#3a6fd8` |
| 의료/보건 | `#ffe4e4` | `#d94f4f` |
| 아동/교육 | `#fff3cc` | `#b8860b` |
| 동물 | `#d4f4e7` | `#277a5c` |
| 긴급구호 | `#ffece0` | `#e05a00` |
| 일반 | `#f0f0f0` | `#666` |

### 타이포그래피

- Font: `-apple-system, BlinkMacSystemFont, 'Noto Sans KR', sans-serif`
- 페이지 제목: `28px / 800`
- 카드 제목: `15–16px / 700–800`
- 본문: `13–14px / 400–500`
- 보조: `11–12px / 400`

### 공통 컴포넌트

- **내비게이션 탭**: pill 형태 (`border-radius: 999px`), 활성 = 검정 배경 흰 글씨
- **카드**: `border-radius: 14–18px`, 흰 배경, hover 시 `box-shadow` 부각
- **버튼 (주요)**: `#1a1a1a` 배경, `border-radius: 10px`
- **서브탭**: 하단 `3px solid` 밑줄 방식

## 4. 페이지별 상세

### 4.1 커뮤니티 — 토론 게시판

**경로**: `community.html` → 서브탭 `💬 토론 게시판`

#### 기능

| 요소 | 설명 |
|------|------|
| 검색바 | 게시글 제목·내용·태그 텍스트 검색 |
| 카테고리 필터 | 전체 / 환경 / 의료 / 아동교육 / 동물 / 긴급구호 / 자유토론 / 기부 후기 |
| 글쓰기 버튼 | 우측 상단 고정 |
| 게시글 카드 | 카테고리 배지, 제목, 미리보기 2줄, 작성자·시간·조회수, 좋아요·댓글 수 |
| HOT 배지 | 좋아요 임계치 초과 시 자동 표시 |
| 페이지네이션 | 하단 번호 버튼 |

#### 데이터 예시 (목업)

- 🔥 **HOT** | 환경 | XRP 기부가 탄소 중립에 실제로 얼마나 기여할 수 있을까요? — ♥47 💬23
- 아동/교육 | 넥스트클래스 재단 장학금 수혜 후기 — ♥38 💬15
- 자유토론 | ETF 뮤음 기능 – 분산 vs 집중 기부 전략 — ♥62 💬41

### 4.2 커뮤니티 — 기부왕 랭킹

**경로**: `community.html` → 서브탭 `🏆 기부왕 랭킹`

#### 레이아웃

```
┌─────────────────────────┬──────────────┐
│  기간 선택 (이번달/주/전체)  │              │
│  시상대 TOP 3            │  카테고리별   │
│  내 순위 카드             │  기부 현황   │
│  전체 랭킹 리스트          │              │
│                          │  업적 배지   │
│                          │              │
│                          │  핫 토론     │
└─────────────────────────┴──────────────┘
```

#### 주요 컴포넌트

| 컴포넌트 | 설명 |
|----------|------|
| **시상대** | 1위(금) 중앙 높이, 2위(은) 왼쪽, 3위(동) 오른쪽 |
| **내 순위 카드** | 검정 그라디언트 배경, 현재 순위 + 상위 % 표시 |
| **랭킹 행** | 순위 번호, 아바타, 닉네임, 누적 XRP, 순위 변동(▲▼) |
| **카테고리 현황** | 환경 38% · 의료 24% · 교육 20% · 동물 12% · 긴급 6% |
| **업적 배지** | 4×2 그리드, 미취득 배지는 grayscale + opacity 처리 |

#### 업적 배지 목록

🌱 첫 기부 · 🔥 7일 연속 · 🌊 환경 수호 · 💯 100 XRP · 👑 기부왕 · 🤝 10명 초대 · 🗳 거버넌스 참여 · 📅 1년 기부

### 4.3 커뮤니티 — 기부 묶음 추천

**경로**: `community.html` → 서브탭 `🎁 기부 묶음 추천`

#### 개념

사용자가 여러 재단을 **배분 비율과 함께 묶어** 커뮤니티에 공유하고,  
다른 기부자들이 해당 묶음에 대해 토론하거나 그대로 장바구니에 담을 수 있는 기능.

#### 기능

| 요소 | 설명 |
|------|------|
| **이번 주 PICK 배너** | 인디고~핑크 그라디언트, 주간 추천 묶음 강조 노출 |
| **정렬 필터** | 인기순 / 최신순 / 토론 활발 / 내가 담은 묶음 |
| **묶음 만들기** | 퍼플 그라디언트 버튼, 커스텀 묶음 생성 진입점 |
| **묶음 카드** | 포함 재단 목록 + 배분 비율 컬러바 + 해시태그 |
| **인라인 토론** | 카드 안에서 토론 댓글 펼치기/접기 + 댓글 입력 |
| **묶음 담기** | 클릭 시 `✔ 담음` 상태 전환, 재클릭으로 취소 |

#### 목업 묶음 카드

| 묶음명 | 구성 재단 | 담은 사람 |
|--------|-----------|-----------|
| 지구를 살리는 3종 세트 🌍 | 그린어스 40% · 블루오션 35% · 커뮤니티펀드 25% | 312명 |
| 아이들을 위한 미래 투자 📚 | 넥스트클래스 50% · 오픈헬스 30% · 애니멀케어 20% | 89명 |
| 환경 올인 전략 🌊 | 그린어스 60% · 블루오션 40% | 134명 |
| 긴급 + 안정 밸런스 ⚖️ | 커뮤니티펀드 35% · 오픈헬스 35% · 그린어스 30% | 62명 |
| 전 카테고리 균등 분배 🌈 | 5개 재단 각 20% | 103명 |

## 5. 전체 내비게이션 구조

```
Truve.
├── 기부 탐색          ← 재단 카드 그리드, 검색, 카테고리 필터, 장바구니 추가
├── 기부 담기          ← 장바구니, 단일/ETF 뮤음 선택, 기부 실행
├── 거버넌스           ← 기부자 투표, 제안 목록
├── 내 기부 현황       ← 누적 기부 내역, 트랜잭션 해시, 임팩트 리포트
├── 커뮤니티           ← (현재 구현)
│   ├── 💬 토론 게시판
│   ├── 🏆 기부왕 랭킹
│   └── 🎁 기부 묶음 추천
├── 서비스 소개        ← Truve 소개, XRPL 연동 설명
└── 지원 센터          ← FAQ, 문의
```
