지갑 연결이 안 될 때

Xaman API 키/시크릿이 서버 환경변수에 들어있는지, API 서버(8787)가 실행 중인지 먼저 확인해 주세요.

기부 실행 후 확인 방법

기부 후 표시되는 TX Hash를 XRPL Testnet Explorer에서 조회하면 검증 상태를 바로 확인할 수 있습니다.

데이터가 이상할 때

목업 단계에서는 localStorage 상태가 남아 있을 수 있어, 장바구니/지갑 상태 초기화 후 다시 테스트해 주세요.

문의 및 피드백

데모 이슈나 개선 요청은 프로젝트 리포지토리 이슈 또는 운영 채널로 전달해 주세요.
